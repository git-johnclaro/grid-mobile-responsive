User Experience

The user experience on the website is not very good. The website is difficult to navigate and it is not clear what the user is supposed to do. There is also no search bar, which makes it difficult for users to find what they are looking for.
User Experience

Content

The content of the website is all over the place. There is a mix of text, images, and videos, and it is not always clear what is for sale and what is not. There is also a lot of text in a foreign language (possibly Norwegian) which could be confusing for users who do not speak that language.
Content

Grid Layout

The website uses a grid layout, but it is not very well-organized. There is a lot of white space between the items, and the items themselves are not all the same size. This can make the page look cluttered and busy.
Grid Layout
